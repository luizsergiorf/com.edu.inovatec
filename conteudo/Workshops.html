WORKSHOPS

Ciência de dados

  • Realização: 11 de junho, das 19h até 22h
  • Duração: 3 horas
  • Vagas: ??

A ciência de dados permite a extração de informações valiosas a partir dos dados, como estamos vivendo na era do Big Data, o aumento dos volumes de dados decorrente das aplicações que se valem das redes sociais e da convergência nas telecomunicações tem gerado um campo promissor para as organizações explorarem e processarem grandes volumes de dados no tempo e no contexto em que atuam. O Workshop terá a função de abrir horizontes para os alunos para a área de ciências de dados em nossa instituição.


Quem é o público-alvo?

O curso é destinado a qualquer pessoa interessada em obter informações sobre ciência de dados e big data.
